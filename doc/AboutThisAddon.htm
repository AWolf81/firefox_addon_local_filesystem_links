Summary This Firefox extension deals with calling the Windows Explorer on the Windows operating system. Target Audience The extensions was created to assist people using local / enterprise intranet sites (e. g. Wikis) where pages exist that link to files that are located on the local drive or on network shares. Details Page context menu entry The extension adds an entry 'Open with Windows Explorer' to the page context menu that works on selected text. Step by Step
  • Select a text portion that contains the path to a local file or folder (e. g. C:\Windows).
  • Right click on the selection and choose 'Open with Windows Explorer' from the context menu.
  • The Windows Explorer is openend with the path selected. If your path was a file, the file will be selected and you only have to press Enter to open the file.
Automatic file link recognition An automatic file link recognition scans for hyperlinks with a file:/// prefix. For each of these hyperlinks a folder icon will be inserted right to the hyperlink. If you click the icon the local file link is opened in Windows Explorer. Supported file links
  • Local disk file links like C:\Windows (file:///C:/Windows)
  • Network share links like \\server\folder\file.png (file://///server/folder/file.png)
